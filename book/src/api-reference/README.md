# API Reference

Complete API documentation for all RustF framework components.

## Available APIs

### [Context API](context.md)
The `Context` struct provides access to request/response functionality:
- Request data access
- Response generation
- Session management
- Flash messages
- Repository data
- Client information

### [Model API](models.md)
Database model operations and query building:
- Model creation and configuration
- Query builder methods
- Relationships
- Transactions
- Migrations

### [Routing API](routing.md)
Route definition and registration:
- Route creation
- Route handlers
- Route parameters
- Route groups
- Auto-discovery

### [Utilities](utilities.md)
Helper functions and utilities:
- String utilities
- Date/time helpers
- Validation functions
- Type conversions
- Common patterns
